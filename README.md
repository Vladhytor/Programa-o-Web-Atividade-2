# Atividade: Página HTML com Formulário de Cadastro
# Disciplina: Programação Web
# Curso: Análise e Desenvolvimento de Sistemas
# Professor: Rafael Marconi Ramos
# Período: 5°

## Descrição
Esta atividade consiste na criação de uma página **HTML** contendo um **formulário de cadastro**, seguindo as regras estabelecidas pelo professor. A página foi criada utilizando apenas recursos básicos de HTML e publicada utilizando **GitHub Pages**.

## Objetivo
Praticar conceitos fundamentais de desenvolvimento web utilizando **HTML puro**, incluindo:
- Estrutura básica de uma página HTML
- Criação de formulários
- Organização de layout com tabelas
- Publicação de páginas na web com GitHub Pages

## Requisitos da Atividade
Durante o desenvolvimento da página, foram seguidas as seguintes regras:

1. Criar uma página HTML.
2. Utilizar as **tags do material do AVA**.
3. Utilizar **tabelas** para organizar o layout.
4. Utilizar **formulários (`form`, `input`, etc.)**.
5. **Não utilizar CSS** nem o parâmetro `style`.
6. **Não utilizar `div`**.
7. Criar um **repositório no GitHub** e realizar o commit da página.
8. Publicar a página utilizando **GitHub Pages**.

## Tecnologias Utilizadas
- **HTML5**
- **Git**
- **GitHub**
- **GitHub Pages**

## Publicação
A página foi publicada utilizando **GitHub Pages**, permitindo o acesso diretamente pelo navegador.

## Observações
- O desenvolvimento foi feito apenas com **HTML**, conforme exigido.
- Não foram utilizados **CSS**, **JavaScript** ou **IA para gerar o trabalho completo**, seguindo as orientações da atividade.
- O código foi escrito utilizando **Bloco de Notas**, conforme solicitado.
